# unstyled-vue

Набор компонентов пользовательского интерфейса с минимальным количеством предопределённых стилей

### Установка пакетов

```sh
yarn install
```

### Интерактивное обновление пакетов

```sh
yarn upgrade-interactive --latest
```

### Запуск демонстрационной версии

```sh
yarn demo
```

### Запуск тестов компонентов в режиме UI mode

```sh
yarn test-ct
```
