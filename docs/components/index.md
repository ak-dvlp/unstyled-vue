# Общий обзор компонентов

В целом компоненты библиотеки не содержат стилей, хотя некоторые стили всё же используются, например, для обеспечения желаемого поведения при работе с программами экранного доступа (screen readers).

Компоненты библиотеки могут быть стилизованы пользователем через соответствующие CSS‑классы.

::: info Информация
В примерах данной библиотеки стили вынесены в отдельные папки. Блоки `<style>` <br />
Vue-компонентов используются для динамического управления стилями через `v-bind`.
:::
