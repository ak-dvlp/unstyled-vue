# Общий обзор компонентов

В целом компоненты библиотеки не содержат стилей, хотя некоторые стили всё же используются, например, для обеспечения желаемого поведения при работе с программами экранного доступа (screen readers).

Компоненты библиотеки стилизуются через CSS-классы, имена которых можно найти в выделенных строках раздела `Шаблон` на странице каждого компонента.

::: info Информация
В примерах данной библиотеки стили вынесены в отдельные папки. Блоки `<style>` <br />
Vue-компонентов используются для динамического управления стилями через `v-bind`.
:::

Обязательные свойства компонентов обозначены красной звёздочкой <span style="color: var(--vp-custom-block-danger-text); font-size: 1.25rem;">\*</span>. Необязательные свойства имеют явно заданные значения по умолчанию для обеспечения типобезопасности, стабильной работы в различных средах сборки и полного соответствия документации.
