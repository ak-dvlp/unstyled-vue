# Общий обзор компонентов

Параметры компонентов перечислены в таблицах. Обязательные параметры отмечены красной звёздочкой <span style="color: var(--vp-custom-block-danger-text); font-size: 1.25rem;">\*</span>. У многих обязательных и у всех необязательных параметров заданы значения по умолчанию. Это помогает обеспечить типобезопасность, а также стабильную работу компонентов в разных средах сборки и в тех случаях, когда динамические данные временно отсутствуют.

Стилизация поддерживается через параметр `classes` объектного типа. Ключи этого объекта указаны в таблице входных параметров. Допустимо использование обычных `CSS-классов`, `Tailwind-классов` или их комбинации.

::: warning Предупреждение
Под `классами` далее в документации понимаются `Tailwind-классы`. <br />
Компоненты раздела `Примеры` не являются обычными стилизованными компонентами библиотеки. Помимо статической и динамической стилизации классами, компоненты в примерах могут содержать дополнительные элементы шаблона или внутреннюю логику. <br />
Пользовательские классы имеют приставку `my`.
:::
