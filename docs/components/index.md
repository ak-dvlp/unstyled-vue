# Общий обзор компонентов

В целом компоненты библиотеки не содержат стилей, хотя некоторые стили всё же используются, например, для обеспечения желаемого поведения при работе с программами экранного доступа (screen readers).

Компоненты библиотеки могут быть стилизованы пользователем через соответствующие CSS‑классы. Классы могут быть размещены:

- в отдельной папке (например, в папке `src/styles/components`) и затем импортированы в файл `main.ts`;
- в блоке `<style>` одностраничных Vue-компонентов.

Примеры, составленные к данной библиотеке, используют первый вариант, размещая стили компонентов в отдельной папке. При этом блок `<style>` одностраничных Vue-компонентов используется для динамического управления стилями через функцию `v-bind`.

::: tip
Вот так по 100 раз на день делай, спина болеть не будет.
:::
