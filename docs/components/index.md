# Общий обзор компонентов

Параметры компонентов перечислены в таблицах. Обязательные параметры отмечены красной звёздочкой <span style="color: var(--vp-custom-block-danger-text); font-size: 1.25rem;">\*</span>. У многих обязательных и у всех необязательных параметров заданы значения по умолчанию. Это помогает обеспечить типобезопасность, а также стабильную работу компонентов в разных средах сборки и в тех случаях, когда динамические данные временно отсутствуют.

В целом компоненты библиотеки не содержат стилей, хотя некоторые стили всё же используются, например, для обеспечения желаемого поведения при работе с программами экранного доступа. Стилизация поддерживается через параметр `classes` объектного типа. Ключи этого объекта указаны в таблице входных параметров, а их расположение в структуре шаблона компонента можно найти в выделенных строках раздела `Шаблон`. Допустимо использование обычных `CSS-классов`, `классов Tailwind` или их комбинации.

::: warning Предупреждение
Компоненты раздела `Примеры` не являются обычными стилизованными компонентами библиотеки. Помимо статической и динамической стилизации служебными классами `Tailwind` и пользовательскими классами (их можно отличить по приставке `my`) определёнными через директиву встроенной темы (`@theme inline`), они также могут содержать некоторые дополнительные элементы шаблона, позволяющие добиться нужного поведения. Тем не менее ниже каждого примера представлены некоторые стили для общего представления.
